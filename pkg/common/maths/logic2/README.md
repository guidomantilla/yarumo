# logic2

Nuevo paquete experimental para lógica proposicional, SAT y un motor de reglas simple.

Estado: Fase 0 — Estructura inicial y esqueletos compilables. Consulte PLAN.md para el roadmap completo.

Objetivo del MVP:
- AST y utilidades de proposiciones (parser mínimo incluido)
- SAT (DPLL) y política de decisión simple
- Motor de reglas con `FactBase`, `Rule` y ejecución a punto fijo con explicación mínima

Este README se actualizará a medida que avancemos por las fases.
